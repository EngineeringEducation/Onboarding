
Install Node  
```shell
brew install node
```

Install Bower  
```shell
npm install bower -g
```

Install Grunt  
```shell
npm install grunt -g
```

Install [postgres.app](http://postgresapp.com/)  

Add psql to your path by opening ~/.bashrc and adding this line (anywhere):
```shell
export PATH=$PATH:/Applications/Postgres.app/Contents/Versions/9.4/bin
```
