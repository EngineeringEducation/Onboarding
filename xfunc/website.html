How to quickly make a website!

Make sure you have a text editor, like Sublime Text or Atom.io

Sign up for Paperplane.io

Go to the Bootstrap homepage

The best way to start a webpage is by starting from a template for example. We're going to copy-paste from the examples area of the documentation.

This Page | Carousel Layout | Blog Layout

Right-click and select "View Source". Copy the entire page, paste it into a text editor.

Replace the links to the stylesheets with links to the official CDNs (basically, free hosted versions of them)

Replace:
<!-- Bootstrap core CSS -->
<link href="../../dist/css/bootstrap.min.css" rel="stylesheet">
With:
<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/css/bootstrap.min.css">

And...

Replace:
<script src="../../dist/js/bootstrap.min.js"></script>>
With:
<!-- Latest compiled and minified JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.4/js/bootstrap.min.js"></script>

Save this file and the unique CSS file from your template

Right click > Save link as > Save the file in the same folder you're saving your homepage
